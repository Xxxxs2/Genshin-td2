# 星海航船塔防原型

这是一个用 Godot 4 制作的“千星奇域式”船塔防原型。玩家控制一艘星海航船移动，船上的武器会自动攻击敌人；每清完一关，玩家从三个随机强化里选择一个，然后进入下一关。

## 运行方式

1. 打开 Godot 4.x。
2. 选择 **Import**。
3. 选择这个文件夹：`/Users/lyh/Documents/Genshin-project`。
4. 打开 `project.godot`。
5. 点击 **Run** 运行。

## 操作

- `WASD`：移动航船。
- 航船会自动攻击附近敌人。
- 清掉当前关全部敌人后，会出现三选一强化。
- 船体生命归零后失败，可以点击 **Restart** 重开。

## 当前原型内容

- 2D 俯视角星海战斗场地。
- 玩家航船作为“会移动的塔”。
- 自动锁敌的星轨主炮。
- 远程弹幕敌人。
- 清图过关流程。
- 每关结束后的三选一强化系统。
- 随关卡递增的敌人数量和强度。
