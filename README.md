# 星海航船塔防原型

这是一个用 Godot 4 制作的“千星奇域式”船塔防原型。玩家驾驶一艘星海航船穿过纵向长海图。海道会弯折、分流和重新汇合，岛上的炮台会在射程内锁定船并攻击；船上的武器会自动攻击范围内炮台。到达海道出口后，玩家从三个随机强化里选择一个，然后进入下一段海道。

## 运行方式

1. 打开 Godot 4.x。
2. 选择 **Import**。
3. 选择这个文件夹：`/Users/lyh/Documents/Genshin-project`。
4. 打开 `project.godot`。
5. 点击 **Run** 运行。

## 操作

- `WASD`：驾驶航船前后左右移动。
- 航船会自动攻击附近炮台。
- 岛屿是硬地形，船不能穿过。
- 炮台有攻击范围，船进入范围后会被锁定攻击。
- 到达上方海道出口后，会出现三选一强化。
- 船体生命归零后失败，可以点击 **Restart** 重开。

## 当前原型内容

- 2D 俯视角纵向长海图与平滑跟随镜头。
- 玩家航船作为“可以航行的塔”。
- 自动锁敌的星轨主炮。
- 岛群构成弯折海道、左右分流、中央险路和汇合点。
- 路线存在“近但危险”与“远但安全”的选择。
- 岛上炮台锁定攻击。
- 到达出口过关，不强制清掉所有炮台。
- 每段海道结束后的三选一强化系统。
- 随海道递增的炮台数量和强度。
- 护盾、环绕星刃、穿透、多弹、减速弹幕等构筑强化。
