# 星海航船塔防原型

这是一个用 Godot 4 制作的“千星奇域式”船塔防原型。星海航船会沿船头方向自动航行，玩家负责控制船头朝向并选择是否加速。海道会弯折、分流和重新汇合，岛上的炮台会在射程内锁定船并攻击；船上的武器会自动攻击范围内炮台。到达海道出口后，玩家从三个随机强化里选择一个，然后进入下一段海道。

## 运行方式

1. 打开 Godot 4.x。
2. 选择 **Import**。
3. 选择这个文件夹：`/Users/lyh/Documents/Genshin-project`。
4. 打开 `project.godot`。
5. 点击 **Run** 运行。

## 操作

- 航船始终沿船头方向自动航行。
- `WASD`：控制船头朝向，可以转弯、横切或回头。
- `Shift`：按住加速，松开后平滑恢复巡航速度。
- 松开方向键后，航船保持当前朝向继续前进。
- 船首主炮只会攻击前方目标，侧舷炮会自动攻击船身两侧目标。
- 获得尾雷强化后，航船会在敌人位于船尾时自动布雷。
- 岛屿是硬地形，船不能穿过。
- 炮台攻击前会显示锁定线；不同颜色分别代表普通、散射和远程炮台。
- 到达上方海道出口后，会出现三选一强化。
- 船体生命归零后失败，可以点击 **Restart** 重开。

## 当前原型内容

- 2D 俯视角纵向长海图与平滑跟随镜头。
- 船头自动前进、全方向转舵和 `Shift` 主动加速。
- 玩家航船作为“可以航行的塔”。
- 自动锁敌的星轨主炮。
- 岛群构成弯折海道、左右分流、中央险路和汇合点。
- 路线存在“近但危险”与“远但安全”的选择。
- 红、蓝、金航线信标提供不同即时奖励。
- 普通锁定炮、近距散射炮、远程狙击炮与关底精英炮台。
- 船首主炮、自动侧舷炮和可构筑的船尾雷。
- 炮击、防御、机动三类强化达到 3 次后触发构筑共鸣。
- 精英炮台封锁终点，击破后才能进入出口。
- 到达出口过关，不强制清掉所有炮台。
- 每段海道结束后的三选一强化系统。
- 随海道递增的炮台数量和强度。
- 护盾、环绕星刃、穿透、多弹、减速弹幕等构筑强化。
