# 星海航船塔防原型

这是一个用 Godot 4 制作的“千星奇域式”船塔防原型。场景会持续从上往下滚动，视觉上表现为航船自动向上远航；玩家只需要左右微调航船位置，船上的武器会自动攻击敌人。每清完一段航线，玩家从三个随机强化里选择一个，然后进入下一段。

## 运行方式

1. 打开 Godot 4.x。
2. 选择 **Import**。
3. 选择这个文件夹：`/Users/lyh/Documents/Genshin-project`。
4. 打开 `project.godot`。
5. 点击 **Run** 运行。

## 操作

- `A/D`：左右微调航船位置。
- 航船会自动攻击附近敌人。
- 清掉当前航线全部敌人后，会出现三选一强化。
- 船体生命归零后失败，可以点击 **Restart** 重开。

## 当前原型内容

- 2D 俯视角竖版星海航线。
- 玩家航船固定在画面下方，作为“缓慢前进的移动塔”。
- 自动锁敌的星轨主炮。
- 从上方进入的远程弹幕敌人。
- 清图过关/航线推进流程。
- 每段航线结束后的三选一强化系统。
- 随航线递增的敌人数量和强度。
- 护盾、环绕星刃、穿透、多弹、减速弹幕等构筑强化。
