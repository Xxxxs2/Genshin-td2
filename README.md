# 星海奇域房间 Roguelike 原型

这是一个用 Godot 4 制作的千星奇域玩法原型。玩家驾驶一艘会沿船头自动航行的星海船，在一个个封闭海域房间之间探索。战斗房需要清理敌船、海怪和岛屿炮台后才会开门；宝藏房、商店、事件房、隐藏房、精英房和首领房提供资源、风险选择与遗物构筑。

玩法采用房间式动作 Roguelike 的通用结构，但不复刻《以撒》的角色、敌人、道具、美术、房间图案或具体数值。

## 运行方式

1. 打开 Godot 4.x。
2. 选择 **Import**。
3. 选择文件夹：`/Users/lyh/Documents/Genshin-project`。
4. 打开 `project.godot`。
5. 点击 **Run**。

## 操作

- 航船始终沿船头方向自动航行。
- `WASD`：改变船头朝向，可以转弯、横切和回头。
- `Shift`：按住加速，松开后恢复巡航速度。
- 船首炮、侧舷炮和解锁后的尾雷会自动攻击。

## 当前玩法

- 单屏封闭海域房间。
- 清怪锁门与清理后开门。
- 敌船、追击海怪和岛屿炮台混合遭遇。
- 起始、战斗、宝藏、精英、商店、事件、隐藏和首领房。
- 星币、钥匙、星爆弹三种探索资源。
- 钥匙门与消耗星爆弹开启的隐藏房。
- 商店购买、祭坛风险选择和房间掉落。
- 首领三选一遗物与下一层循环。
- 炮击、防御、机动三类构筑共鸣。
